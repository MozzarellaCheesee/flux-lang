Напиши полноценную документацию в README.md файл для репозитория github про язык мой программирования Flux

В языке реализованны базовые типы:
int8 int16 int32 int64 int128
uint8 uint16 uint32 uint64 uint128
float double
&str (как в Rust) string
size_t usize_t
bool

Так же реализованны все базовые операции
+ - * / % ~ += -= *= /= %= ++x x++ --x x-- 
Операторы сравнения:
== != < > <= >= 
Присваивание:
= 
Ссылки/указатели
& . 
Пунктуация:
( ) [ ] { } : , ; -> => 
Препроцессор:
#import
Служебные:
_
Побитовые операторы:
&& | || ^ << >> 
Унарные операторы:
!
Ключевые слова:
fnc - Функция
let - Объявление переменной
return - Возвращение 
for - цикл
whie - цикл
if else - Условия
continue - Следущая итерация цикла
break - Остановить цикл
true false - булевые операнды
match - конструкция match
