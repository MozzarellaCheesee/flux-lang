## Быстрый старт

```flux
fnc main() -> int32 {
    println("Hello, Flux!");
    return 0;
}
```

```bash
fluxc.exe hello.flx          # собирает hello.exe
fluxc.exe hello.flx --emit-cpp  # показывает сгенерированный C++
```

## Структура проекта

```
flux-lang/
├── src/
│   ├── lexer/          # Токенизация
│   ├── preprocessor/   # #import
│   ├── parser/         # Парсинг
│   ├── sema/           # Семантический анализ
│   ├── codegen/        # Генерация C++
│   ├── complete/       # Автодополнение
│   ├── lsp/            # Language Server Protocol
│   └── main.cpp        # Точка входа компилятора
├── include/flux/
│   ├── lexer/
│   ├── preprocessor/
│   ├── parser/
│   ├── ast/            # Узлы AST
│   ├── sema/
│   ├── codegen/
│   ├── common/         # Диагностика, SourceLocation
│   └── stdlib/         # Встроенные модули
├── Examples/           # Примеры программ
└── docs/               # Эта документация
```
