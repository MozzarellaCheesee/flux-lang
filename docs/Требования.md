| Инструмент         | Минимальная версия | Назначение                  |
| ------------------ | ------------------ | --------------------------- |
| CMake              | 3.28               | Система сборки              |
| C++ компилятор     | C++20              | Компиляция самого fluxc     |
| GCC / Clang / MSVC | любой с C++20      | Компиляция программ на Flux |

## Сборка компилятора

```bash
# Из корня репозитория
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build

# Или для VS Code (build-vsc директория)
cmake --build build-vsc --config Debug
```

Результат: `build/fluxc.exe` (Windows) или `build/fluxc` (Linux/macOS).

## Использование компилятора

### Базовая компиляция

```bash
fluxc.exe hello.flx
# → создаёт hello.exe рядом с исходником
```

### Указать имя выходного файла

```bash
fluxc.exe hello.flx -o build/hello
fluxc.exe hello.flx --output build/hello
```

### Посмотреть сгенерированный C++

```bash
# --no-compile: только сгенерировать .cpp, не компилировать
fluxc.exe hello.flx --no-compile
# → создаёт hello.cpp

# --emit-cpp: скомпилировать И сохранить .cpp рядом с exe
fluxc.exe hello.flx --emit-cpp
```

### Выбрать C++ компилятор

```bash
fluxc.exe hello.flx --compiler gcc    # принудительно g++
fluxc.exe hello.flx --compiler clang  # принудительно clang++
fluxc.exe hello.flx --compiler msvc   # принудительно cl.exe
```

По умолчанию: автоматически выбирает первый найденный в PATH: `g++` → `clang++` → `cl`.

### Дополнительные флаги

```bash
fluxc.exe hello.flx --print-ast   # вывести AST в stdout
fluxc.exe hello.flx -v            # подробный вывод (verbose)
fluxc.exe hello.flx --verbose

fluxc.exe --version   # версия компилятора
fluxc.exe --help      # справка
```

## Коды возврата

| Код | Этап | Значение |
|-----|------|----------|
| 0 | — | Успешно |
| 1 | Лексер | Лексическая ошибка |
| 2 | Препроцессор | Ошибка импорта |
| 3 | Парсер | Синтаксическая ошибка |
| 5 | Семантика | Ошибка типов / объявлений |
| 6 | CodeGen | Не удалось записать .cpp |
| 7 | — | C++ компилятор не найден |
| 8 | — | Ошибка C++ компиляции |

## Специальные режимы

### LSP-сервер

```bash
fluxc.exe --lsp
```

Запускает Language Server Protocol сервер. Используется VS Code расширением для подсветки синтаксиса, диагностики и автодополнения в реальном времени.

### Автодополнение

```bash
fluxc.exe --complete <line> <col> <file>
```

Возвращает JSON-массив подсказок для позиции `(line, col)` в файле. Используется VS Code расширением.

```bash
# Пример:
fluxc.exe --complete 5 12 main.flx
# → [{"label":"println","kind":3,"detail":"built-in function"}, ...]
```

## Первая программа

Создайте файл `hello.flx`:

```flux
fnc main() -> int32 {
    println("Hello, Flux!");
    return 0;
}
```

Скомпилируйте и запустите:

```bash
fluxc.exe hello.flx
./hello.exe          # Windows: hello.exe
```

## Программа с stdlib

```flux
#import <math>
#import <string>

fnc main() -> int32 {
    let x: double = 144.0;
    println("sqrt(144) = ", sqrt(x));

    let s: string = "  hello world  ";
    println(str_trim(str_to_upper(s)));

    return 0;
}
```
